# Choi Junseok Portfolio

최준석의 학부연구생 포트폴리오 웹사이트입니다. CAD, ANSYS, Python과 Manufacturing AI에 관한 학습 및 연구 자료를 소개합니다.

## GitHub Pages 배포

1. GitHub에서 비어 있는 새 저장소를 만듭니다. 예: `portfolio`
2. 이 폴더의 파일을 저장소의 `main` 브랜치에 업로드합니다.
3. 저장소의 **Settings → Pages**에서 **Deploy from a branch**를 선택합니다.
4. Branch를 `main`, Folder를 `/ (root)`로 선택하고 저장합니다.
5. GitHub가 안내하는 Pages 주소에서 사이트를 확인합니다.

PDF 자료는 `assets/pdfs/` 폴더에 넣은 뒤, `index.html`의 아카이브 카드 링크를 해당 파일 경로로 바꾸면 됩니다.
